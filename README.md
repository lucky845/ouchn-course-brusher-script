# 国家开放大学实验学院自动刷课脚本

基于 Vue 3 + TypeScript 重构的油猴脚本，适用于国家开放大学实验学院的 Moodle 平台。（个人自用）

## 功能特点

### 刷课功能
- 🎓 **自动识别学习项目**：自动检测课程页面中的视频、文档等学习项目
- 📊 **实时进度显示**：显示当前学习位置和已刷数量
- ⏱️ **本次刷课计时**：记录本次刷课持续时间
- ⚡ **速度控制**：支持正常/快速/低调三种模式
- 🔒 **防息屏功能**：防止电脑/手机息屏，方便挂机刷课
- 🛡️ **防检测机制**：模拟真人行为，随机滚动页面
- 💾 **会话持久化**：刷新页面后保留刷课统计和已刷数量
- 📍 **边缘吸附**：悬浮窗拖拽后可自动吸附到屏幕左/右侧

### 课程管理
- 📚 **多学期课程展示**：支持展示所有学期课程，当前学期默认展开
- 📊 **学期统计卡片**：显示已完成/待办/总进度等统计信息
- 🎯 **粘性标题**：滚动时学期标题自动吸顶，方便快速定位
- 🔄 **课程刷新**：一键刷新课程列表
- 📖 **快捷导航**：快速跳转到刷课中心

### 答题功能
- 📝 **题目提取**：一键提取答题页面所有题目（单选/多选/判断/完形填空等）
- 🔍 **题型智能检测**：优先识别题型文字（.summary-sub-title），再降级 class 匹配
- 📋 **选项完整提取**：兼容 OUCHN 和 Moodle 两种 DOM 结构，支持拆分多行选项
- 🔢 **多小题自动拆分**：综合题/应用题自动拆分为独立子题
- 📎 **一键复制**：提取的题目自动格式化并复制到剪贴板

### 悬浮窗
- 🎨 **现代化 UI**：紫色主题控制面板，美观大方
- 🖱️ **拖拽定位**：支持在屏幕上自由拖拽
- 📌 **边缘吸附**：拖拽结束后自动贴附到最近一侧边缘
- 💾 **位置记忆**：自动保存面板位置，刷新后恢复上次位置
- 📱 **F12 兼容**：窗口大小变化后自动调整位置

## 技术栈

- **Vue 3** Composition API + `<script setup>` 语法
- **TypeScript** 类型安全
- **Vite** 构建工具
- **油猴脚本** 用户脚本打包

## 安装方法

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)（油猴）
2. 点击 [安装脚本](https://github.com/lucky845/ouchn-course-brusher-script/raw/main/ouchn-course-brusher-vue.user.js)
3. 按提示完成安装

## 使用方法

### 刷课

1. 登录国家开放大学实验学院 Moodle 平台
2. 进入课程页面
3. 点击悬浮的 🎓 按钮打开控制面板
4. 选择速度模式（正常/快速/低调）
5. 点击「开始」按钮启动自动刷课

### 课程管理

1. 进入学习中心页面
2. 点击悬浮的 📊 按钮打开课程管理面板
3. 查看各学期课程列表和统计信息
4. 点击「去学习」按钮跳转到课程页面
5. 点击「刷新」按钮更新课程列表

### 提取题目

1. 进入答题页面
2. 点击悬浮的 📝 按钮
3. 题目自动提取并复制到剪贴板

## 控制面板说明

### 刷课面板

| 功能 | 说明 |
|------|------|
| 本次 N 个 | 本次会话已完成的课程数（跨页面保持） |
| 本次时间 | 本次刷课持续时长 |
| 速度模式 | 正常/快速/低调三种速度可选 |
| 视频倍速 | 仅快速模式下生效，支持 1x/1.5x/2x/3x |
| 防息屏 | 开启后防止息屏 |
| 防检测 | 开启后模拟真人行为 |

### 课程管理面板

| 功能 | 说明 |
|------|------|
| 已完成 | 当前学期已完成的课程数 |
| 待办 | 当前学期待完成的课程数 |
| 总进度 | 当前学期总体完成百分比 |
| 学期列表 | 所有学期课程，可折叠展开 |
| 刷新 | 刷新课程列表 |
| 刷课中心 | 跳转到刷课中心页面 |

### 答题面板

| 功能 | 说明 |
|------|------|
| 智能提取 | 兼容 OUCHN 和 Moodle DOM 结构 |
| 暴力提取 | 扩大搜索范围，提高提取成功率 |
| 超强提取 | 激进策略，适合复杂页面 |
