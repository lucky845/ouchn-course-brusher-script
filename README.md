# 国家开放大学实验学院自动刷课脚本

## 功能介绍

这是一个自动刷完学习项目的脚本，适用于国家开放大学实验学院的 Moodle 平台。

### 主要功能
- 自动识别并处理学习项目（视频、文档、网页等）
- 自动播放视频并在视频结束后继续下一个项目
- 智能跳过需要手动操作的项目（如作业、测验等）
- 可拖拽的悬浮控制窗口，支持吸附到屏幕边缘
- 支持浏览器缩放时自动调整位置
- 保存用户设置和窗口位置

## 安装方法

1. 确保浏览器已安装脚本管理器：
   - Tampermonkey (推荐用于 Chrome/Edge/Firefox)
   - Greasemonkey (Firefox)
   - Violentmonkey (Chrome/Edge/Firefox)

2. 点击以下链接安装脚本：
   - [安装脚本](https://github.com/yourusername/ouchn-course-brusher/raw/main/ouchn-course-brusher.user.js)

3. 安装后，刷新课程页面即可看到悬浮控制窗口。

## 使用方法

1. **启用/禁用脚本**：点击悬浮窗口中的开关
2. **开始刷课**：点击"开始刷课"按钮
3. **停止刷课**：点击"停止刷课"按钮
4. **拖拽窗口**：拖动悬浮窗口到合适位置，会自动吸附到屏幕边缘
5. **调整面板位置**：点击按钮时，面板会根据窗口位置自动调整到左侧或右侧

## 注意事项

1. 脚本仅自动处理不需要手动操作的学习项目
2. 对于作业、测验等需要手动操作的项目，脚本会自动跳过
3. 请确保网络连接稳定，以确保脚本正常运行
4. 脚本在后台运行，不会影响您的正常浏览

## 技术说明

- 使用 JavaScript 编写
- 基于 Tampermonkey API
- 支持现代浏览器
- 响应式设计，适配不同屏幕尺寸
